# O Grito das Palavras 

Site gerador de livros de poesia inspirado em grandes autores e obras de arte.

---

## Estrutura do Projeto

```plaintext
ogrito/
├── index.html                ← Página principal
├── css/
│   └── style.css             ← Estilos do site
├── js/
│   └── app.js                ← Lógica principal
└── assets/
    └── covers/
        ├── dali.png          ← A Persistência da Memória — Salvador Dalí
        ├── klimt.png         ← O Beijo — Gustav Klimt
        ├── girassois.png     ← Os Girassóis — Van Gogh
        ├── monalisa.png      ← Mona Lisa — Leonardo da Vinci
        ├── noite_estrelada.png ← Noite Estrelada — Van Gogh
        ├── guernica.png      ← Guernica — Picasso
        └── abaporu.png       ← Abaporu — Tarsila do Amaral
```

---

## Como Executar o Projeto

### Método 1 — Live Server (Recomendado)

1. Abra a pasta `ogrito/` no VS Code
2. Instale a extensão **Live Server**
3. Clique com o botão direito em `index.html`
4. Selecione **"Open with Live Server"**
5. O site será aberto automaticamente no navegador

---

### Método 2 — Abrir diretamente no navegador

1. Abra a pasta `ogrito/`
2. Dê duplo clique em `index.html`

> ⚠️ Algumas imagens locais podem não funcionar corretamente ao abrir pelo `file://`.
> Recomenda-se utilizar o Live Server ou outro servidor local.

## Funcionalidades

- Fundo artístico e dinâmico
- Login simulado com Google
- Geração automática de livros de poesia
- Temas disponíveis:
  - Romance
  - Político
  - Existência
- Sistema de capas rotatórias com obras de arte famosas
- Biblioteca pessoal salva com `localStorage`
- Leitura inline dos poemas
- Download em PDF via impressão do navegador
- Biografia dos autores ao final das páginas
- Proteção parcial contra cópia de conteúdo
